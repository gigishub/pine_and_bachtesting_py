## PXAccessor class[](https://github.com/polakowo/vectorbt/blob/993ceca7116fc8e55f4cd3a36fe43d83dab62b27/vectorbt/px_accessors.py#L69-L89 "Jump to source")¶
    
    
    PXAccessor(
        obj,
        **kwargs
    )
    

Accessor for running Plotly Express functions.

Accessible through `pd.Series.vbt.px` and `pd.DataFrame.vbt.px`.

**Usage**
    
    
    >>> import pandas as pd
    >>> import vectorbt as vbt
    
    >>> vbt.settings.set_theme('seaborn')
    
    >>> pd.Series([1, 2, 3]).vbt.px.bar()
