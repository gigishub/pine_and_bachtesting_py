# generators module[](https://github.com/polakowo/vectorbt/blob/993ceca7116fc8e55f4cd3a36fe43d83dab62b27/vectorbt/labels/generators.py "Jump to source")¶

Basic look-ahead indicators and label generators.

You can access all the indicators either by `vbt.*` or `vbt.labels.*`.

Create a combination of multiple [FIXLB](https://vectorbt.dev/api/labels/generators/#vectorbt.labels.generators.FIXLB "vectorbt.labels.generators.FIXLB") indicators using function `comb_func`.

  * Inputs: `close`
  * Parameters: `n`

`comb_func` must accept an iterable of parameter tuples and `r`. Also accepts all combinatoric iterators from itertools such as `itertools.combinations`. Create a combination of multiple [FMAX](https://vectorbt.dev/api/labels/generators/#vectorbt.labels.generators.FMAX "vectorbt.labels.generators.FMAX") indicators using function `comb_func`.

  * Inputs: `close`
  * Parameters: `window`
  * Outputs: `fmax`



`comb_func` must accept an iterable of parameter tuples and `r`. Also accepts all combinatoric iterators from itertools such as `itertools.combinations`. Pass `r` to specify how many indicators to run. Pass `short_names` to specify the short name for each indicator. Set `run_unique` to True to first compute raw outputs for all parameters, and then use them to build each indicator (faster).

Other keyword arguments are passed to [FMAX.run()](https://vectorbt.dev/api/labels/generators/#vectorbt.labels.generators.FMAX.run "vectorbt.labels.generators.FMAX.run").

Create a combination of multiple [FSTD](https://vectorbt.dev/api/labels/generators/#vectorbt.labels.generators.FSTD "vectorbt.labels.generators.FSTD") indicators using function `comb_func`.

  * Inputs: `close`
  * Parameters: `window`, `ewm`

`comb_func` must accept an iterable of parameter tuples and `r`. Also accepts all combinatoric iterators from itertools such as `itertools.combinations`. Set `run_unique` to True to first compute raw outputs for all parameters, and then use them to build each indicator (faster).

Get array of local extrema denoted by 1 (peak) or -1 (trough), otherwise 0.

Two adjacent peak and trough points should exceed the given threshold parameters.

        comb_func=itertools.combinations,
        run_unique=True,
        short_names=None,

    )
    

Create a combination of multiple [LEXLB](https://vectorbt.dev/api/labels/generators/#vectorbt.labels.generators.LEXLB "vectorbt.labels.generators.LEXLB") indicators using function `comb_func`.

  * Inputs: `close`
  * Parameters: `pos_th`, `neg_th`
  * Outputs: `labels`



`comb_func` must accept an iterable of parameter tuples and `r`. Also accepts all combinatoric iterators from itertools such as `itertools.combinations`. Pass `r` to specify how many indicators to run. Pass `short_names` to specify the short name for each indicator. Set `run_unique` to True to first compute raw outputs for all parameters, and then use them to build each indicator (faster).

Other keyword arguments are passed to [LEXLB.run()](https://vectorbt.dev/api/labels/generators/#vectorbt.labels.generators.LEXLB.run "vectorbt.labels.generators.LEXLB.run").

        run_unique=True,

    )
    

Create a combination of multiple [MEANLB](https://vectorbt.dev/api/labels/generators/#vectorbt.labels.generators.MEANLB "vectorbt.labels.generators.MEANLB") indicators using function `comb_func`.

  * Inputs: `close`
  * Parameters: `window`, `ewm`
  * Outputs: `labels`



`comb_func` must accept an iterable of parameter tuples and `r`. Also accepts all combinatoric iterators from itertools such as `itertools.combinations`. Pass `r` to specify how many indicators to run. Pass `short_names` to specify the short name for each indicator. Set `run_unique` to True to first compute raw outputs for all parameters, and then use them to build each indicator (faster).

### run_combs class method[](https://github.com/polakowo/vectorbt/blob/993ceca7116fc8e55f4cd3a36fe43d83dab62b27/vectorbt/indicators/factory.py#L1-L1210 "Jump to source")¶
    
    
    TRENDLB.run_combs(
        close,
        pos_th,
        neg_th,
        mode=Default(0),
        r=2,
        param_product=False,
        comb_func=itertools.combinations,
        run_unique=True,
        short_names=None,
        hide_params=None,
        hide_default=True,
        **kwargs
    )
    

Create a combination of multiple [TRENDLB](https://vectorbt.dev/api/labels/generators/#vectorbt.labels.generators.TRENDLB "vectorbt.labels.generators.TRENDLB") indicators using function `comb_func`.

  * Inputs: `close`
  * Parameters: `pos_th`, `neg_th`, `mode`
  * Outputs: `labels`



`comb_func` must accept an iterable of parameter tuples and `r`. Also accepts all combinatoric iterators from itertools such as `itertools.combinations`. Pass `r` to specify how many indicators to run. Pass `short_names` to specify the short name for each indicator. Set `run_unique` to True to first compute raw outputs for all parameters, and then use them to build each indicator (faster).

Other keyword arguments are passed to [TRENDLB.run()](https://vectorbt.dev/api/labels/generators/#vectorbt.labels.generators.TRENDLB.run "vectorbt.labels.generators.TRENDLB.run").
